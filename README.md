# ISTRUCTIONS
Virtual environment
```bash
$ virtualenv env
$ source env/bin/activate
```

Install dependencies
```bash
$ pip install -r requirements.txt
```

Runserver
```bash
python manage.py runserver
```

List project-url available: <http://localhost:8000>
